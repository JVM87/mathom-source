# Mathom Source

Public installation source for **Mathom**, a Kodi addon that helps save videos resolved by compatible Kodi add-ons so they can be watched offline later.

Source URL for Kodi:

```text
https://jvm87.github.io/mathom-source/
```

Current package:

```text
plugin.video.mathom-1.0.2.zip
```

## English

### What Is Mathom?

**Mathom** is a Kodi video add-on designed to save compatible resolved video streams for offline viewing. It works as a lightweight offline download manager for Kodi: when another compatible add-on resolves a playable video URL, Mathom can capture that final link and download the video in the background.

Mathom is useful if you want to:

- Save a video for later viewing.
- Download compatible streams in the background.
- Pause, resume, and cancel active downloads.
- Keep a local offline media folder inside Kodi.
- Avoid keeping Kodi blocked by a foreground download window.

### Features

- Kodi video add-on for offline viewing.
- Background video downloads.
- Active download list with progress, speed, and ETA.
- Pause, resume, and cancel controls.
- Local downloads section for playback from Kodi.
- Compatible with direct HTTP/HTTPS video links resolved by supported add-ons.

### How To Install In Kodi

1. Open Kodi.
2. Go to **Settings -> File manager -> Add source**.
3. Add this URL:

   ```text
   https://jvm87.github.io/mathom-source/
   ```

4. Give the source a name, for example `Mathom`.
5. Go to **Add-ons -> Install from zip file**.
6. Open the `Mathom` source.
7. Select `plugin.video.mathom-1.0.2.zip`.
8. Open Mathom and configure your download folder in the add-on settings.

### Important Notes

Mathom is a technical download-management tool. It does not provide video catalogs, channels, movies, series, accounts, debrid services, or hosted content. Users are responsible for ensuring they have the right to download and store any content they save.

## Español

### Qué Es Mathom

**Mathom** es un add-on de vídeo para Kodi pensado para guardar vídeos resueltos por add-ons compatibles y verlos más tarde sin conexión. Funciona como un gestor ligero de descargas offline para Kodi: cuando otro add-on compatible resuelve una URL final reproducible, Mathom puede capturar ese enlace y descargar el vídeo en segundo plano.

Mathom puede ser útil si quieres:

- Guardar un vídeo para verlo más tarde.
- Descargar streams compatibles en segundo plano.
- Pausar, reanudar y cancelar descargas activas.
- Mantener una carpeta local de vídeos offline dentro de Kodi.
- Evitar que Kodi quede bloqueado por una ventana de descarga en primer plano.

### Funciones

- Add-on de vídeo para Kodi orientado a visualización offline.
- Descargas de vídeo en segundo plano.
- Lista de descargas activas con progreso, velocidad y tiempo estimado.
- Controles para pausar, reanudar y cancelar.
- Sección de descargas locales para reproducir desde Kodi.
- Compatible con enlaces directos HTTP/HTTPS resueltos por add-ons compatibles.

### Cómo Instalar En Kodi

1. Abre Kodi.
2. Ve a **Ajustes -> Gestor de archivos -> Añadir fuente**.
3. Añade esta URL:

   ```text
   https://jvm87.github.io/mathom-source/
   ```

4. Ponle un nombre a la fuente, por ejemplo `Mathom`.
